The scenario is contained in the file VS02-CL07.PDF and contains all player and GM handouts.

File VS02-LS07.PDF is a log sheet for the scenario.  This log sheet is voluntary for players at this

filling out the log sheet are available online at www.virtualseattle.net or by emailing 
bitrunner@cfl.rr.com
