## Features

- **Visual Node Editor** - Drag and drop interface for building systems
- **Custom IC Placement** - Choose exactly where each IC goes
- **Connection Management** - Define how nodes connect
- **Security Ratings** - Set individual node security levels
- **Save/Load Systems** - Build a library of reusable systems
- **Export Options** - Print or share your designs

## How to Use

1. **Create Nodes** - Add nodes to your system
2. **Connect Nodes** - Draw connections between nodes
3. **Add IC** - Place IC in strategic locations
4. **Set Security** - Configure security ratings
5. **Test & Refine** - Review your system layout
6. **Save** - Export for use in your game

## Perfect For

- **Detailed Prep** - Build complex systems for major runs
- **Unique Locations** - Create memorable Matrix environments
- **Teaching** - Show players how Matrix systems work
- **Reusability** - Build once, use multiple times

## Design Tips

- Start with the paydata location and work backwards
- Place IC at chokepoints and critical nodes
- Consider multiple entry points for variety
- Balance challenge with player capabilities
- Add flavor with node descriptions
