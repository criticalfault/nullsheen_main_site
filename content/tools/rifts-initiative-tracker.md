## Features

- **Actions Per Melee** - Track multiple attacks per round
- **Initiative Order** - Automatic sorting by initiative roll
- **Add/Remove Combatants** - Manage NPCs and PCs dynamically
- **Action Tracking** - Mark actions as they're used
- **Round Counter** - Keep track of melee rounds

## How to Use

1. **Add Combatants** - Enter names and initiative rolls
2. **Set Actions** - Input attacks per melee for each combatant
3. **Track Combat** - Click through actions and rounds
4. **Manage Participants** - Add or remove as needed
5. **Reset** - Start fresh for the next encounter

## Rifts Combat Basics

Rifts uses a unique combat system where:
- Initiative determines action order
- Characters have multiple attacks per melee round
- Actions include attacks, dodges, and parries
- Tracking can get complex with many combatants

This tracker simplifies all of that!
